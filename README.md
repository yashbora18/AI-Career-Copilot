# AI Based Resume Analyzer 🚀

AI-powered Resume Analyzer web application built using Flask, Groq AI, and MySQL. The system analyzes resumes and provides skill evaluation, missing skills, roadmap suggestions, interview questions, and recommended job roles.

---

## 📌 Features

- User Signup & Login
- Resume Upload (PDF/DOCX)
- AI Resume Analysis
- Resume Score Generation
- Missing Skills Detection
- Personalized Roadmap
- Interview Questions
- Recommended Job Roles
- Download Report as PDF
- User History Tracking
